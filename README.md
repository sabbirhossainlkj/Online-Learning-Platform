# 🎓 SkillSphere - Online Learning Platform

SkillSphere is a modern online learning platform where users can explore courses, learn new skills, and grow their careers. Built with Next.js and Tailwind CSS, it provides a clean UI and smooth user experience.


## ✨ Features

* 🎥 Hero Banner Section (Modern UI)
* 🔥 Popular Courses (Top-rated courses)
* 📘 Learning Tips (Study + Time Management)
* 👨‍🏫 Top Instructors Section
* 📚 All Courses Page (Dynamic data fetch)
* 🔍 Course Details Page (Protected Route)
* 🔐 Authentication System (Login / Signup)
* 🎯 Responsive Design (Mobile + Desktop)



## 🔒 Protected Route

* Course Details page is accessible only for logged-in users
* If not logged in → redirect to Login page
* After login → redirect back to the intended page



## 🛠️ Tech Stack

* ⚛️ Next.js (App Router)
* 🎨 Tailwind CSS
* 🧩 React Components
* 🔗 Fetch API
* 🎯 HeroUI (UI Components)
* 🎭 React Icons


## 📦 Installation & Setup

```bash
git clone https://github.com/your-username/skillsphere.git
cd skillsphere
npm install
npm run dev
```
